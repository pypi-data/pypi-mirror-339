## ✨ 核心特性

### 🚀 核心功能
- 开箱即用的 OpenAI/DeepSeek/Gemini...... 协议支持
- 可独立运行的基本聊天机器人
- 支持群聊/私聊双模式
- AT触发与智能上下文管理
- 戳一戳消息交互支持
- 多模型热切换能力
- 多角色热切换能力

### 🧩 扩展体系
- 模块化协议适配器架构
- 开放插件API接口
- SuggarMatcher扩展处理器
- 自定义消息解析引擎

### 🛠️ 高级能力
- 伪人模式（随机概率自动回复）
- 消息撤回缓解机制
- 异常日志推送至管理群
- 会话生命周期控制
- Token智能管理策略
